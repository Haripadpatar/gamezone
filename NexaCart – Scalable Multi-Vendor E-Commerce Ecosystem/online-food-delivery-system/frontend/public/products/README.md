# Product Images Folder

Place your product image files here (e.g. `veg_chowmein.jpg`, `chicken_biryani.jpg`, etc.).
The application maps the product names to lowercase files with spaces replaced by underscores (e.g. `Veg Chowmein` -> `veg_chowmein.jpg`).
If a local image is missing, the application will automatically fall back to the remote database URL, and then the default food placeholder.
